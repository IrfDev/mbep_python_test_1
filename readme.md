# Book of address


## Contact schema
Contact:
- Name
- Last name
- Tel
- Mail
- Street
- Ext
- Int
- Town
- Area
- City
- State



## Instructions

Create a program that:

- Save contact information in a json file
- Retrieve a contact
- Retrieve the amount of contacts
- Delete contacts 
- Retrieve all the contacts

### Bonus:
- Retrieve a specified number of contacts
